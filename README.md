## Instructions
Clone this repository:
```
git clone https://github.com/BarbosaRT/2D-Raycast.git
```
Access the directory:
```
cd 2D-Raycast
```
Now install the dependencies:
```
pip install -r requirements.txt
```
And run:
```
python3 demo.py
```
